Tables

Tables are a nice way to organize a lot of data. Expertiza default use bootstrap table library to style our tables as easily as possible

Sr. No. talbe Class Name Guide Code
1 Table The base class, the quick and default way to include tables table class="table"
2 Borderd Table Adds borders on all sides of the table and cells table class="table table-bordered"
3 Striped Table Adds zebra-stripes to a table table class="table table-striped"
4 Hover Table Adds a hover effect (grey background color) on table rows table class="table table-hover"
