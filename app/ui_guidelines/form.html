Text

  1. General Font name: verdana,arial,helvetica,sans-serif

  2. Headings
    Headings/ Main title of page should be given in <h2> or <h3> tags

  3. Font Size
    • Menu bar
      font size of 13 px
      line height 30px
      (will be automatically applied)
    • Content Headings
      Use h2 or h3 so that it will be uniform throughout in expertiza
    • Content Subheadings / sub-tab titles
      font size of 1.2 em
      line height 18px
      (will be automatically applied)
    • Table data
      font size of 13 px
      line height 1.428 em
      (will be applied autamatically)

  4. Color
    • Menu bar
      color: #EEEEEE; or color: #FFFFFF; //for menubar with red background
    • Other titles/ text
      color: #333;
      (it will be applied automatically)
    • Content Subheadings / sub-tab titles
      font size of 1.2 em
      line height 18px
      (will be automatically applied)
    • Text on red buttons
      color: #fff;
